# Data Instructions

Dataset: PlantVillage from Kaggle.
Link: https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset

If you use Colab:
1. You must create a folder in Google Drive containing your kaggle.json file to allow Kaggle to download the dataset.

2. The notebook contains the code for Google Colab to connect to your Google Drive, where the data will be stored.

3. Once connected to Google Drive, it will use the kaggle.json file to download and extract the dataset.

4. Do not upload the data to the repository.
